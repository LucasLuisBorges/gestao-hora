## Tecnologias Utilizadas
- React
- Styled-Components

## Instalação
Siga as etapas abaixo para configurar e executar o projeto localmente.

1. Clone o repositório
```bash
git clone https://github.com/LucasLuisBorges/gestao-hora.git
cd gestao-hora
```

2. Instale as dependências
```bash
npm install
# ou
yarn install
```
